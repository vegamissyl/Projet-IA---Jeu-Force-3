Ce projet permet de jouer au jeu "Force 3" selon 3 modes :
- Joueur contre Joueur
- Joueur contre IA
- IA contre IA

L'implémentation de l'IA a été réalisé à partir des algorithmes Min/Max et Alpha-Bêta

Pour lancer le jeu il suffit de cliquer sur l'exécutable "Accueil.exe" du dossier "build"
Chemin de l'exécutable :  Projet IA - Jeu Force 3\build\exe.win-amd64-3.8\Accueil.exe
